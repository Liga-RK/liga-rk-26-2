# Sorteio oficial dos grupos

Os dois aplicativos funcionam sem servidor e não são incluídos no `dist` público:

- `sorteio-elite.html`
- `sorteio-ascensao.html`

## Uso durante a transmissão

1. Abra o HTML da divisão no navegador.
2. Clique em **Roleta** e depois em **Roletar**.
3. O resultado entra automaticamente no grupo e o sistema avança de A até D.
4. O quarto time de cada pote entra automaticamente no Grupo D.
5. Ao terminar os quatro potes, clique em **Sortear posições**.
6. Clique em **Baixar JSON** para guardar a composição final.

Recarregar ou fechar a página reinicia toda a sessão. Use o botão de seta circular no cabeçalho para reiniciar manualmente.

O JSON contém os grupos, os slots `A1` até `D4`, tags, nomes, logos, potes e o histórico do sorteio.
